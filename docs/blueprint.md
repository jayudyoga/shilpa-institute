# **App Name**: TuitionFlow

## Core Features:

- Admin Dashboard: A centralized dashboard providing quick navigation links to management sections and displaying high-level summary statistics of teachers, students, and payments.
- Teacher Management: Complete CRUD (Create, Read, Update, Delete) functionality for managing teacher profiles, including their subjects, contact details, and activity status.
- Student Management: Complete CRUD (Create, Read, Update, Delete) functionality for student profiles, including their grade, parent contact information, and enrolled classes.
- Class Management: Complete CRUD (Create, Read, Update, Delete) functionality for defining tuition classes, specifying class names, assigned teachers, monthly fees, and schedules.
- Payment Processing: A dedicated interface for admin users to select a student, choose an enrolled class, input the amount paid, and record the payment in the system.
- Payment History & Reporting: View and filter a detailed log of all recorded payments, allowing for tracking by student, class, date range, or amount for financial oversight.
- Digital Receipt Generation: Upon successful payment recording, automatically generate a well-formatted digital receipt (PDF) containing student name, class, amount paid, payment date, and a unique receipt number.

## Style Guidelines:

- The primary color, conveying professionalism and learning, is a muted yet distinctive blue (#2D63D3). It serves as the dominant accent for interactive elements and key headings.
- The background features a soft, almost white hue derived from the primary blue (#F0F2F5). This provides a clean, understated canvas that promotes clarity and readability.
- An accent color of vibrant purple (#6952E0) is used to draw attention to critical calls-to-action or significant data points, providing an energetic counterpoint while maintaining an analogous harmony.
- A single typeface, 'Inter' (sans-serif), is chosen for all text, ensuring a modern, objective, and highly legible experience across both headlines and body copy.
- Utilize a consistent set of clean, scalable vector-based icons that visually represent each administrative function and data type, enhancing usability without visual clutter.
- Adopt a structured, dashboard-centric layout with clear separation between content areas, employing generous whitespace to maintain an uncluttered and efficient user experience.
- Incorporate subtle and swift animations for interactions like button clicks, form submissions, and data loading states, providing visual feedback without distracting from the user's tasks.
